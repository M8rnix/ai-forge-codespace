# Step 1: Configure Custom Agent

In this step, you will create your first AI-augmented tool: a **Custom Agent**. This agent will act as your lead developer, following your specific instructions to build the project.

### Instructions

1.  **Open the Agent Picker**: Click on the Copilot icon in the Chat view and select **Configure Custom Agent**.
    ![ConfigureCustomAgent](../assets/ConfigureCustomAgent.png)

2.  **Follow the Wizard**: Use the menu wizard at the top of your screen to set up the agent.
    ![Wizard](../assets/Wizard.png)

3.  **Define the Persona**: Give the agent instructions that define it as an expert developer.

    **Example Instructions:**
    ```markdown
    You are an EXPERT developer, and always make sure to build and validate the application after making adjustments.
    ```

4.  **Save the Agent**: Complete the wizard to save the agent in the `.github/agents` folder.

---
*Once you have created the agent, return to the Forge Master and say "Done".*
