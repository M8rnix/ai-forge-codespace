## 👋 Intro

You’re going to build a small C# console app, but the real skill is the workflow: use GitHub Copilot Chat to plan the steps, generate code, run it, read the errors, and iterate until it’s solid. By the end, you’ll have a working CLI and a repeatable “prompt → code → run → fix → refine” loop you can reuse on real projects.
 

## 🛠️ Pre-Requisites

Basic C# / .NET CLI familiarity

## How to Start

Ready to ignite the forge? Open your **GitHub Copilot Chat** window.

And run the starting prompt by typing: <code>/Start-Forge</code>

Github Copilot will guide you through the Forge

## 🛠️ Maintenance & Support

Sometimes the forge needs a little cleaning. Use the toggles below if you run into any issues.
