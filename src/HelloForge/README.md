HelloForge
=========

Small .NET 10 console app that prints "Hello World".

Build and run (requires .NET 10 SDK installed):

```bash
# create via dotnet CLI (optional)
# dotnet new console -o src/HelloForge -n HelloForge --framework net10.0

# build
dotnet build src/HelloForge

# run
dotnet run --project src/HelloForge
```

Expected output:

```
Hello World
```
