## 🔥 Starting your forge

<div align="right">
 

</div>

Once you have your own repository setup:

1. **Open Codespaces:** Click the green **Code** button at the top right of your new repository.
2. **Create Codespace:** Select the **Codespaces** tab and click **Create codespace on main**.
3. **Wait for Setup:** GitHub will configure your environment based on the included `.devcontainer` settings.

</p>

To get started, open Github Copilot Chat and send the message: **`/forge-start-introduction`**.  
Our include agent will guide you through some of the basics.  

## 🛠️ What's Inside

Your forge comes pre-configured for immediate use:

- **Dev Container:** A fully-featured environment.
- **VS Code Extensions:** Extensions for AI development and debugging are pre-installed.
- **Introduction agents:** The forge agents will help you find your first Forge.
